# Polzunov.Feast, панель администратора

## Локально

### Требования

- node.js
- npm

### Начальные действия

- `npm ci` - установка необходимых зависимостей
- `npm run generate-api` - генерация клиента из openapi.yaml

### Запуск

`npm run dev`

## Docker

### Требования

- docker
- docker-compose

### Запуск

`docker-compose up`

Приложение будет доступно по адресу http://localhost:9000
